# 2026 软件工程总复习 · 详细解析网站

## 本地打开

双击 `start-server.bat`，浏览器访问 **http://localhost:8080**

同一 WiFi 下的同学可用局域网地址访问（bat 窗口会显示 IP）。

## 分享给别人

### 方式一：局域网（最快，同一 WiFi）

1. 双击 `start-server.bat`
2. 点击网站右上角 **🔗 分享**
3. 把局域网地址发给同学

### 方式二：公网永久链接（推荐）

1. 在 [GitHub](https://github.com) 注册并新建仓库（如 `software-engineering-review`）
2. 上传本文件夹**全部文件**
3. 仓库 **Settings → Pages → Build and deployment → Source** 选择 **GitHub Actions**
4. 等待 1～3 分钟，访问：

```
https://你的GitHub用户名.github.io/仓库名/
```

把这个链接发给任何人即可访问，无需安装。

## 自定义背景

点击右上角 **🖼 背景**：

- 上传本地图片（自动压缩、自适应铺满屏幕）
- 或粘贴网络图片 URL
- 调节遮罩透明度保证文字可读
- 设置保存在浏览器，刷新后仍有效
