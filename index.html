② 局域网分享（同一 WiFi）

③ 公网永久链接（推荐）

  1. 在 GitHub 新建仓库，上传本文件夹全部文件
  2. 仓库 Settings → Pages → Source 选 GitHub Actions
  3. 推送后几分钟获得地址：
    https://你的用户名.github.io/仓库名/

部署后任何人打开链接都能学习，无需安装任何东西。
